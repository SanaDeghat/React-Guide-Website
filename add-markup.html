Add markup and styles

Writing markup with JSX

The markup syntax you’ve seen in React examples is called JSX. It’s basically “HTML-ish syntax inside JavaScript.”

JSX is stricter than HTML:

  • You must close tags like <br />
  • A component can’t return multiple JSX elements unless they’re wrapped in one parent

Here’s an example using a Fragment wrapper (<> ... </>):

function AboutPage() { return ( <> <h1>About</h1> <p>Hello there.<br />How do you do?</p> </> ); }

Adding styles

In React, you specify a CSS class with className. It works like HTML’s class:

<img className="avatar" />

Then you write the CSS rules in a separate CSS file:

/* In your CSS */ .avatar { border-radius: 50%; }

React doesn’t force one styling system. You can do plain CSS (like this site), CSS modules, Tailwind, styled-components, etc.
