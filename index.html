What is React?

Have you ever wanted to make a cool website... but HTML started looking like a mess of <div>s? Or the mix of JavaScript, HTML, and CSS files was just becoming too confusing? React is a JavaScript library for building user interfaces using small, reusable pieces called components.

Many big companies use React (like Discord and Facebook) and so can you! This guide is meant to make your journey into React a little bit easier.
