Create and nest components

React apps are made out of components. A component is a piece of the UI (user interface) that has its own logic and appearance.

A component can be as small as a button, or as large as an entire page. The main idea is: instead of one huge HTML file, you build lots of small reusable pieces.

Step 1: Make a component

React components are JavaScript functions that return markup (usually JSX). Here’s a tiny component called MyButton:

function MyButton() { return ( <button>I'm a button</button> ); }

Step 2: Nest it inside another component

Now that you’ve declared MyButton, you can use it inside another component just like an HTML tag:

export default function MyApp() { return ( <div> <h1>Welcome to my app</h1> <MyButton /> </div> ); }

Important: Components must start with a capital letter

Notice that <MyButton /> starts with a capital letter. That’s how you know it’s a React component.

  • React components must start with a capital letter: MyButton, ProfileCard, ShoppingList
  • HTML tags are lowercase: <div>, <button>, <img>

If you accidentally name your component starting with lowercase (like myButton), React will treat it like a regular HTML element and your component won’t render correctly.
