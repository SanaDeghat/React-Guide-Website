Share data between components

Sometimes you want components to share data and always update together. The main pattern for this in React is called lifting state up.

The problem: state is isolated per component

If you render MyButton twice, each one has its own count. Clicking one button doesn’t update the other.

The fix: move the state to the closest shared parent

To make both buttons show the same count, move state up into the parent component (MyApp), then pass it down as props.

Step 1: put the state in the parent

import { useState } from 'react'; export default function MyApp() { const [count, setCount] = useState(0); function handleClick() { setCount(count + 1); } return ( <div> <h1>Counters that update together</h1> {/* next step: pass count + click handler down */} </div> ); }

Step 2: pass state + handler as props

export default function MyApp() { const [count, setCount] = useState(0); function handleClick() { setCount(count + 1); } return ( <div> <h1>Counters that update together</h1> <MyButton count={count} onClick={handleClick} /> <MyButton count={count} onClick={handleClick} /> </div> ); }

Step 3: read props in the child

function MyButton({ count, onClick }) { return ( <button onClick={onClick}> Clicked {count} times </button> ); }

Now clicking either button updates the parent’s state, and React passes the new value down to both buttons. That’s “lifting state up.”
