Display data

JSX lets you put markup into JavaScript. Curly braces { } let you “escape back” into JavaScript so you can display variables.

Show a value in JSX

return ( <h1> {user.name} </h1> );

Use variables inside attributes

In JSX attributes, use curly braces instead of quotes when you want a JavaScript value.

return ( <img className="avatar" src={user.imageUrl} /> );

A full example (text + image + inline styles)

You can also use expressions inside curly braces (like string concatenation). And style={{ ... }} is a plain JS object.

const user = { name: 'Hedy Lamarr', imageUrl: 'https://i.imgur.com/yXOvdOSs.jpg', imageSize: 90, }; export default function Profile() { return ( <> <h1>{user.name}</h1> <img className="avatar" src={user.imageUrl} alt={'Photo of ' + user.name} style={{ width: user.imageSize, height: user.imageSize }} /> </> ); }
