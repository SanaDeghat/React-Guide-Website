Completing the game

You’ve built the basic ingredients: squares, clicks, and a board. Now we’re going to turn it into an actual game: players take turns, nobody can cheat by overwriting squares, and the app can announce a winner!

Lifting state up

Right now, if every Square keeps its own state, the Board has no idea what’s going on. It’s like trying to run a group project where everyone keeps their notes on a napkin and refuses to share.

In React, when multiple child components need to share the same data, we store that shared state in the closest common parent. This is called lifting state up.

Step 1: Store the board in one array

In Board, create a squares state array with 9 slots (one per square). Each slot is either 'X', 'O', or null (empty).

import { useState } from 'react'; function Square({ value }) { return <button className="square">{value}</button>; } export default function Board() { const [squares, setSquares] = useState(Array(9).fill(null)); return ( <> <div className="board-row"> <Square value={squares[0]} /> <Square value={squares[1]} /> <Square value={squares[2]} /> </div> <div className="board-row"> <Square value={squares[3]} /> <Square value={squares[4]} /> <Square value={squares[5]} /> </div> <div className="board-row"> <Square value={squares[6]} /> <Square value={squares[7]} /> <Square value={squares[8]} /> </div> </> ); }

At this point your board is empty again. That is what we want. We just moved the “memory” into the parent.

Step 2: Let squares tell the board they were clicked

Squares can’t directly change Board’s state (state is private to the component that owns it). So we pass a function down as a prop, and the square calls it when clicked.

function Square({ value, onSquareClick }) { return ( <button className="square" onClick={onSquareClick}> {value} </button> ); }

Now add a handleClick(i) function in Board, and pass it to each square. Important: we pass a function, we don’t call it during render, so we use an arrow function.

export default function Board() { const [squares, setSquares] = useState(Array(9).fill(null)); function handleClick(i) { const nextSquares = squares.slice(); nextSquares[i] = 'X'; setSquares(nextSquares); } return ( <> <div className="board-row"> <Square value={squares[0]} onSquareClick={() => handleClick(0)} /> <Square value={squares[1]} onSquareClick={() => handleClick(1)} /> <Square value={squares[2]} onSquareClick={() => handleClick(2)} /> </div> <div className="board-row"> <Square value={squares[3]} onSquareClick={() => handleClick(3)} /> <Square value={squares[4]} onSquareClick={() => handleClick(4)} /> <Square value={squares[5]} onSquareClick={() => handleClick(5)} /> </div> <div className="board-row"> <Square value={squares[6]} onSquareClick={() => handleClick(6)} /> <Square value={squares[7]} onSquareClick={() => handleClick(7)} /> <Square value={squares[8]} onSquareClick={() => handleClick(8)} /> </div> </> ); }

Step 3: Take turns (X, then O, then X…)

Right now every click places an X, which is very pro-X and not very fair to O. Let���s track whose turn it is with a boolean called xIsNext.

export default function Board() { const [xIsNext, setXIsNext] = useState(true); const [squares, setSquares] = useState(Array(9).fill(null)); function handleClick(i) { const nextSquares = squares.slice(); if (xIsNext) { nextSquares[i] = 'X'; } else { nextSquares[i] = 'O'; } setSquares(nextSquares); setXIsNext(!xIsNext); } // ... }

Step 4: No overwriting

Try clicking the same square twice. Oops, it gets overwritten! We don't want that. Fix it by returning early if the square is already filled.

function handleClick(i) { if (squares[i]) { return; } const nextSquares = squares.slice(); nextSquares[i] = xIsNext ? 'X' : 'O'; setSquares(nextSquares); setXIsNext(!xIsNext); }

Step 5: Declare a winner (preferably dramatically)

Time for the judge. Add a helper function called calculateWinner.

function calculateWinner(squares) { const lines = [ [0, 1, 2], [3, 4, 5], [6, 7, 8], [0, 3, 6], [1, 4, 7], [2, 5, 8], [0, 4, 8], [2, 4, 6], ]; for (let i = 0; i < lines.length; i++) { const [a, b, c] = lines[i]; if (squares[a] && squares[a] === squares[b] && squares[a] === squares[c]) { return squares[a]; } } return null; }

Now prevent moves after someone has already won, and show a status message:

export default function Board() { const [xIsNext, setXIsNext] = useState(true); const [squares, setSquares] = useState(Array(9).fill(null)); function handleClick(i) { if (calculateWinner(squares) || squares[i]) { return; } const nextSquares = squares.slice(); nextSquares[i] = xIsNext ? 'X' : 'O'; setSquares(nextSquares); setXIsNext(!xIsNext); } const winner = calculateWinner(squares); let status; if (winner) { status = 'Winner: ' + winner; } else { status = 'Next player: ' + (xIsNext ? 'X' : 'O'); } return ( <> <div className="status">{status}</div> {/* ... board rows here ... */} </> ); }
