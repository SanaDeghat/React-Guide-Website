Setup for the tutorial

In order to be able to work with React first we need to complete a few prerequisits.

Step 1: Install Node.js

Install Node.js (LTS version is best). This gives you npm, your app wont work without this.

Step 2: Start fresh with Vite (recommended)

Vite is a modern tool that sets up React quickly and doesn’t make your laptop sound like a jet engine. 

npm create vite@latest tic-tac-toe -- --template react
                cd tic-tac-toe
                npm install
                npm run dev

Then open the folder in VS Code and visit the local link it prints (usually http://localhost:5173).

If anything explodes (it happens), don’t get stuck here forever. Use an online editor, ask for help in the help channel (if you havnt yet join slack btw!!) or come back to local setup later when you’re feeling powerful.
