Render conditions and lists

Conditional rendering

In React there’s no special “if in JSX” syntax. You just use normal JavaScript patterns.

Option 1: if/else outside JSX

let content; if (isLoggedIn) { content = <AdminPanel />; } else { content = <LoginForm />; } return ( <div> {content} </div> );

Option 2: ternary inside JSX

<div> {isLoggedIn ? ( <AdminPanel /> ) : ( <LoginForm /> )} </div>

Option 3: && (when you don’t need an else)

<div> {isLoggedIn && <AdminPanel />} </div>

Rendering lists

To render a list, you’ll usually use map() to transform an array into an array of elements.

const products = [ { title: 'Cabbage', id: 1 }, { title: 'Garlic', id: 2 }, { title: 'Apple', id: 3 }, ]; const listItems = products.map(product => <li key={product.id}> {product.title} </li> ); return ( <ul>{listItems}</ul> );

Notice the key on each <li>. Keys help React keep track of items when you insert, delete, or reorder them.

List example with conditional styling

const products = [ { title: 'Cabbage', isFruit: false, id: 1 }, { title: 'Garlic', isFruit: false, id: 2 }, { title: 'Apple', isFruit: true, id: 3 }, ]; export default function ShoppingList() { const listItems = products.map(product => <li key={product.id} style={{ color: product.isFruit ? 'magenta' : 'darkgreen' }} > {product.title} </li> ); return ( <ul>{listItems}</ul> ); }
